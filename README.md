# JogoAdivinhacaoC3
Trabalho final C3
Jogo das Cores — Tema Dark Gamer

Um jogo interativo desenvolvido em HTML, CSS e JavaScript, onde o jogador deve
adivinhar uma cor secreta escolhida aleatoriamente pelo sistema, utilizando até 3
tentativas.

O projeto utiliza manipulação do DOM, eventos, validação e estilização com tema neon
futurista.
***** Funcionalidades
Sorteio aleatório de cores
Sistema de tentativas (3 chances)
Interface dark mode com neon azul
Mensagens de erro, acerto e fim de jogo
Mudança dinâmica da cor de fundo quando acerta
Botão “Jogar Novamente” para reinício

***** Tecnologias Utilizadas
Tecnologia Função
HTML Estrutura da página
CSS Estilização (tema dark + neon)
JavaScript Lógica do jogo e DOM

** Estrutura do Projeto
Jogo_Dark_Gamer/
├─ index.html
├─ style.css
└─ script.js

*** Regras do Jogo
1. Uma cor é sorteada aleatoriamente.
2. O jogador deve digitar o nome da cor em português.
3. Se acertar:
o O fundo muda para a cor correta.
o Uma mensagem de sucesso aparece.
4. Se errar:
o O sistema avisa e mostra tentativas restantes.
5. Após 3 erros:
o O jogo revela a cor correta.
o O botão reiniciar é exibido.
